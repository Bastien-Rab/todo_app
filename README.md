# 📝 To-Do List en Python (débutant)

Petit projet pour apprendre Python : une application de gestion de tâches dans le terminal.

## Fonctionnalités

- Afficher la liste des tâches
- Ajouter une tâche
- Supprimer une tâche
- Sauvegarder dans un fichier `tasks.txt`

## Lancer le programme

```bash
python main.py
